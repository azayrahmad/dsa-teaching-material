Bagian 2 · Konsep Dasar · 25 menit
💡 Solusi: Array 2 Dimensi
int nilai[5][3];
1Indeks pertama (5): Jumlah Baris (Mahasiswa)
2Indeks kedua (3): Jumlah Kolom (Mata Kuliah)

nilai[2][1] berarti mahasiswa ke-3, mata kuliah ke-2 (karena indeks mulai dari 0).

Bagian 3 · Traversal & Contoh · 30 menit
🔄 Traversal 2D:
Nested Loop
for(int i = 0; i < 2; i++){         // Loop Baris
    for(int j = 0; j < 3; j++){     // Loop Kolom
        cout << matriks[i][j] << " ";
    }
    cout << endl;
}
1Loop luar (i) mengontrol baris mana yang sedang diakses.
2Loop dalam (j) mengunjungi setiap kolom pada baris tersebut.

Bagian 3 · Template Umum
🛠 Template Umum Traversal
for(int i = 0; i < jumlahBaris; i++){
    for(int j = 0; j < jumlahKolom; j++){
        // Proses elemen matriks[i][j]
    }
}
Pola ini adalah fondasi untuk hampir semua manipulasi data 2D.

📥 Input Data 2D

        cout << "Nilai mhs " << i+1
             << " matkul " << j+1 << ": ";

}

Bagian 4 · Pengolahan Baris
📊 Rata-rata Tiap Mahasiswa
for(int i = 0; i < 5; i++){
    int total = 0;
    for(int j = 0; j < 3; j++){
        total += nilai[i][j];
    }
    cout << "Rata-rata mhs " << i+1
         << ": " << (float)total / 3 << endl;
}
KONSEP PENTING
Variabel total harus di-reset menjadi 0 setiap kali ganti baris (mahasiswa baru).
14 / 20
Bagian 4 · Pengolahan Kolom
📉 Rata-rata Tiap Mata Kuliah
for(int j = 0; j < 3; j++){ // Loop Kolom di luar
    int total = 0;
    for(int i = 0; i < 5; i++){ // Loop Baris di dalam
        total += nilai[i][j];
    }
    cout << "Rata-rata matkul " << j+1
         << ": " << (float)total / 5 << endl;
}
Bandingkan dengan slide sebelumnya. Di sini kita menukar urutan loop untuk memproses data per kolom.

Bagian 5 · Operasi Matriks · 15 menit
➕ Penjumlahan Dua Matriks
// Syarat: Ukuran A dan B harus sama
for(int i = 0; i < 3; i++){
    for(int j = 0; j < 3; j++){
        C[i][j] = A[i][j] + B[i][j];
    }
}
▸Elemen dijumlahkan sesuai posisi baris dan kolom yang sama.
16 / 20
Bagian 5 · Operasi Matriks
🔃 Transpose Matriks

Mengubah baris menjadi kolom dan sebaliknya.

for(int i = 0; i < 3; i++){
    for(int j = 0; j < 3; j++){
        transpose[j][i] = matriks[i][j];
    }
}
▸Perhatikan pertukaran indeks: [i][j] menjadi [j][i].
